## Documentation standards

*Every finished work should have:*

- [ ] High-resolution photograph (natural light, straight-on, no distortion)
- [ ] Detail photograph if texture or material is significant
- [ ] Entry in Works database with all fields complete
- [ ] Title (even if "Untitled [series] [number]")
- [ ] Dimensions in cm
- [ ] Medium and support documented exactly
- [ ] Date completed (month and year minimum)
