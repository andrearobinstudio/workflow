# Weekly Log

> *One entry per week. Sunday's studio and Monday's thinking in one place. Over time, this becomes the truest record of your practice.*

---

## How to use

Create one file per week, named: `YYYY-WW.md` (e.g. `2025-W18.md`)
Use the [template →](../../04-resources/templates/weekly-log-entry.md)
